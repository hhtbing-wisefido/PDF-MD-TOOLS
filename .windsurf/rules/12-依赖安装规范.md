## ?? 核心规则

### 遇到依赖安装慢或无响应时

**必须立即执行**（不等待超过30秒）：

1. **优先使用SOCKS5代理**：
   `ash
   pip install --proxy socks5://127.0.0.1:4000 包名
   `

2. **备选：使用国内镜像**：
   `ash
   pip install -i https://pypi.tuna.tsinghua.edu.cn/simple 包名
   `
