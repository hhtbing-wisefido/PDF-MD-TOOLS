# Dependency Installation Rules

**Rule Type**: MANDATORY  
**Enforcement**: STRICT

---

## Core Rules

### When pip install is slow or hangs

**Execute immediately** (do not wait more than 30 seconds):

1. **Use SOCKS5 proxy first**:
   ```bash
   pip install --proxy socks5://127.0.0.1:4000 PACKAGE
   ```

2. **Fallback: Use China mirror**:
   ```bash
   pip install -i https://pypi.tuna.tsinghua.edu.cn/simple PACKAGE
   ```

## AI Requirements

1. If pip hangs > 30s -> switch to proxy/mirror immediately
2. Do NOT wait, do NOT ask, just switch
3. Priority: SOCKS5 proxy > Tsinghua > Others

---

## AI Self-Check Before pip install

- Did I use proxy? (--proxy socks5://127.0.0.1:4000)
- Or did I use mirror? (-i https://pypi.tuna.tsinghua.edu.cn/simple)
- If neither -> FIX THE COMMAND NOW!

**FORBIDDEN**:
- pip install without proxy/mirror
- Wait > 30s without switching
- Ask user whether to switch

---

**AI must execute proactively, do not wait for user!**
